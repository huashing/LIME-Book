コンストラクタ ↑
BML.Ajax([url] [, options])
↑
Ajax(同期/非同期HTTP通信インタフェース)用クラス．
optionsによるパラメタの初期化処理を行い，urlが指定されている場合はrequest処理を行う.
実体はBML.Ajax-Ajax．
引数
String [url] リクエスト先URL
Object [options] リクエストに関するオプション
String [method='POST'] 送信時のメソッド(POST|GET)
Boolean [asynchronous=true] 同期/非同期通信フラグ(true:非同期)
String [contentType='application/x-www-form-urlencoded'] リクエストのMIMEタイプ
String [overrideMimeType=''] 応答のMIMEタイプの指定．ただし，transportがoverrideMimeTypeメソッドを有しない場合は無視する．
String [encoding='UTF-8'] 送信時の文字コードの指定．
Object [parameters={}] URLのクエリ部(GET)あるいはbody(POST)に設定する引数群．プロパティ名をkey，プロパティ値をvalueとする．
String [postBody=''] POST時のbodyに設定する文字列．options.parametersより優先される．
Funcion [onSuccess=undefined] リクエスト成功時に実行するコールバック関数の指定．コールバック関数の引数は[BML.Ajax-Response]．
Funcion [onFailure=undefined] リクエスト失敗時に実行するコールバック関数の指定．コールバック関数の引数は[BML.Ajax-Response]．
Funcion [onUninitialized=undefined] リクエスト初期化前に実行するコールバック関数の指定．コールバック関数の引数は[BML.Ajax-Response]．
Funcion [onLoading=undefined] リクエスト応答受信中に実行するコールバック関数の指定．コールバック関数の引数は[BML.Ajax-Response]．
Funcion [onLoaded=undefined] リクエスト応答受信完了時に実行するコールバック関数の指定．コールバック関数の引数は[BML.Ajax-Response]．
Funcion [onInteractive=undefined] リクエスト応答解析後に実行するコールバック関数の指定．コールバック関数の引数は[BML.Ajax-Response]．
Funcion [onComplete=undefined] リクエスト完了時に実行するコールバック関数の指定．コールバック関数の引数は[BML.Ajax-Response]．
Funcion [on"HTTPStatusCode"=undefined] readyState変化時のHTTPのステータスコードに対応するコールバック関数の指定．コールバック関数の引数は[BML.Ajax-Response]．
Funcion [onError=undefined] 例外発生時に実行するコールバック関数の指定．コールバック関数の引数は[BML.Ajax, Exception]．
