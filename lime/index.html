Welcome to LIME Sample page



This page gives links to various content made with LIME (ITU-T H.762),"Lightweight Interactive Multimedia Environment", a multimedia environment for Smart TV, IPTV, DTV, etc. These links show that LIME is a version of HTML. Well, XHTML to be precise. In fact, this page is written in LIME, with enough details filled in so that it can be interpreted as HTML. You can right-click and open the source to see it for yourself.
